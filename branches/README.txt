This directory contains experimental branches, used to test new features and developments.
As of 04.09.2013, the following branches exist:

- jAER_1.5: cleanup/refactor test-ground. Features up-to-date JAR libraries and JOGL 2.0, as well as improved libusb support. Runs on Java7. (llongi, tobidelbruck)
- jAER_2.0: rewrite of the jAER framework, with new cleaned-up architecture, implementing support for multiple inputs/sources and outputs/renderers. Uses latest JavaFX8 and Java8 technology. (llongi)
- multisrc: experimental multiple sources support. (peteroconnor)
