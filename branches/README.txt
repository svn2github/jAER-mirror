This directory contains experimental branches, used to test new features and developments.

- jAER2: rewrite of the jAER framework, with new cleaned-up architecture, implementing support for multiple inputs/sources and outputs/renderers. Uses latest JavaFX and JDK8 technology. (llongi)
- multisrc: experimental multiple sources support. (peteroconnor)
- newui-refactor: cleanup/refactor test-ground. Features up-to-date JAR libraries and JOGL 2.0. (llongi, tobidelbruck)
