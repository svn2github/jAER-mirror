Holds utiiities for jAER project.

ServoTest.cmd - launches ServoTest class that can be used to test USB servo interface
CypressFX2EEProm.cmd - launches class to program EEProms on CypressFX2 boards - DANGER only for experts. You can corrupt your EEPROM requiring soldering a new one!
jAERLauncher.cmd and jAERLauncher.exe - starts a small (Java) Windows tray application that launches jAERViewer when compatible hardware is plugged in
