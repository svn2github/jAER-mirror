This is Cypress FX2LP 8051 firmware for the CochleaAMS1c chip boards.
Use Keil uVision2 to develop.
Install Keil uVision to c:\keil.
Install the FX2 development kit to c:\cypress.

This board adds ADC to convert the cochlea analog outputs.
The PCB design is in the CAVIAR repository
https://svn.ini.uzh.ch/repos/avlsi/CAVIAR/wp3/Boards/CochleaAMS1c_USB
The SmartPDF cochleaams1c.pdf of the PCB design should help with design.
