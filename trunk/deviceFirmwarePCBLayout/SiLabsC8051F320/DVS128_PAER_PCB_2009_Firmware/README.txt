This is firmware for DVS128_PAER board designed by Angel Jiménez <ajimenez@atc.us.es>  and Tobi Delbruck (tobi@ini.phys.ethz.ch).

It is also firmwre for Tmpdiff128 board with SiLabs chip designed by Patrick Lichtsteiner for the CAVIAR project.
The board type is #define'ed in USB_Main.h.
This firmwre is used with the USBIO driver on the host side using the DVS128 AEChip class and the SiLabsC8051F320_USBIO_DVS128 hardware interface.
