USB hardware interfaces under linux.

INSTALLATION:

linux requires the usbfs to be running.

BEFORE running jAER under linux run this in a shell (or at linux startup):

sudo mount -o devmode=0666 -t usbfs usbfs /proc/bus/usb

Root privileges are required to run this sudo command.

See the linux jAER startup script for more information. (jaer/trunk/jAERViewer.sh)
