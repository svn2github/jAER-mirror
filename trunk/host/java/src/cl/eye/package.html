Code Laboratories CL Eye Playstation Eye camera wrapper. Runs only under Windows XP/7 32/64 bit.

* See CodeLaboratories DL Eye Multicam C++ API * for the native API.

The Code Laboratories driver is not open-source, and you cannot directly download the driver to install it from an INF file in Windows. Instead you download a program that downloads and installs the driver.

This version was tested under CodeLabs Devices/PS3Eye driver version 4.0.2.1017, dated 10/17/2010.

The driver is also available in drivers/ps3EyeCodeLabs for direct installation.

The CLCamera class is the basic hardware interface that configures and opens the PS3Eye camera and delivers frames from it. It is a JNI interface to the native windows DLL.
