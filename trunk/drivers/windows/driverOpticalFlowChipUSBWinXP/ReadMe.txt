This folder contains a customized thesycon driver for the 
SiLabs C8051F320-based USB interface to the 
OpticalFlowBoard from Pit Gebbers with Alan Stocker's motion18 chip.

To install this driver, plug in the device, then choose 
"Don't search for driver", then navigate to the driverUSBWinXP 
folder and Windows will find the driver.

If you have problems, email tobi@ini.phys.ethz.ch
